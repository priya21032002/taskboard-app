# TaskBoard - Full Stack Task Management App

A modern full-stack task management application built using React, Tailwind CSS, and FastAPI.

## Features

- Add, complete, and delete tasks
- Priority-based task management
- Progress tracking with live updates
- Task filtering (All / Active / Completed)
- Responsive cyberpunk-inspired UI
- REST API integration with FastAPI
- JSON-based lightweight persistence

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- Python

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /tasks | Fetch all tasks |
| POST | /tasks | Create a new task |
| PUT | /tasks/{id} | Toggle task completion |
| DELETE | /tasks/{id} | Delete a task |
